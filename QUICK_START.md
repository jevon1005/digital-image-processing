# 快速开始指南 - Quick Start Guide

## 5分钟入门 - Get Started in 5 Minutes

### 步骤 1: 系统要求检查 - Step 1: Check System Requirements

确保你的系统满足以下要求：
- MATLAB R2018b 或更高版本
- Image Processing Toolbox

检查方法 Check method:
```matlab
ver  % 查看MATLAB版本和已安装的工具箱
```

### 步骤 2: 启动应用 - Step 2: Launch the Application

在MATLAB命令窗口输入：
```matlab
ImageProcessingApp
```

应用程序窗口将会打开，显示双面板界面：
- 左侧：控制面板（功能按钮）
- 右侧：图像显示区域（原始图像 vs 处理后图像）

### 步骤 3: 加载测试图像 - Step 3: Load a Test Image

1. 点击"加载图像 - Load Image"按钮
2. 选择一张图像文件（支持 .jpg, .png, .bmp, .tif）
3. 图像将显示在左侧的"原始图像"面板中
4. 底部会显示图像信息（尺寸、通道数等）

💡 **提示 Tip**: 如果你没有测试图像，可以使用MATLAB内置的示例图像：
```matlab
% 创建测试图像
img = imread('peppers.png');
imwrite(img, 'test_image.png');
```

### 步骤 4: 尝试基本功能 - Step 4: Try Basic Features

#### 试试直方图均衡化 Try Histogram Equalization:
1. 加载一张对比度较低的图像
2. 点击"直方图均衡化"按钮
3. 观察右侧处理后的图像 - 对比度应该明显增强

#### 试试图像反转 Try Image Negative:
1. 点击"图像反转"按钮
2. 图像将变成负片效果

#### 试试平滑滤波 Try Smoothing Filter:
1. 点击"平滑滤波"按钮
2. 图像会变得更平滑（减少噪声但会模糊）

### 步骤 5: 保存结果 - Step 5: Save Your Results

1. 处理完图像后，点击"保存图像 - Save Image"按钮
2. 选择保存位置和文件名
3. 选择文件格式（JPEG, PNG, BMP）
4. 点击保存

## 常用操作流程 - Common Workflows

### 流程 1: 增强低对比度图像
```
1. 加载图像
2. 点击"直方图均衡化"
3. 如果效果不够，尝试"幂次变换"并调整Gamma值
4. 保存结果
```

### 流程 2: 去除图像噪声
```
1. 加载图像
2. 点击"添加椒盐噪声"（模拟噪声）
3. 点击"中值滤波"（去除噪声）
4. 对比原图和处理结果
```

### 流程 3: 锐化模糊图像
```
1. 加载模糊图像
2. 点击"锐化滤波"
3. 或者使用"高通滤波"
4. 观察边缘增强效果
```

### 流程 4: 彩色图像增强
```
1. 加载彩色图像
2. 点击"彩色增强"
3. 颜色会变得更鲜艳
4. 保存结果
```

## 常见问题 - FAQ

### Q1: 点击按钮没反应？
**A**: 确保已经加载了图像。大多数功能需要先加载图像。

### Q2: 为什么彩色功能不能用？
**A**: 某些功能只适用于彩色图像（如"彩色增强"），确保加载的是RGB图像。

### Q3: 如何在处理结果上继续处理？
**A**: 先保存处理结果，然后重新加载保存的图像。每次处理都基于原始图像。

### Q4: Gamma值设置多少合适？
**A**: 
- Gamma < 1 (如0.5): 增亮图像
- Gamma = 1: 无变化
- Gamma > 1 (如2.0): 变暗图像

### Q5: 哪个滤波器最适合去除噪声？
**A**: 
- 高斯噪声 → 使用均值滤波
- 椒盐噪声 → 使用中值滤波

## 学习路径建议 - Suggested Learning Path

### 初学者 Beginner:
1. 从基本操作开始（加载、保存）
2. 尝试简单的变换（反转、直方图均衡化）
3. 了解图像信息的含义

### 中级 Intermediate:
1. 学习空间滤波（平滑、锐化）
2. 理解Gamma变换的效果
3. 尝试噪声添加和去除

### 高级 Advanced:
1. 深入频率域处理（FFT、频域滤波）
2. 研究色彩空间转换
3. 组合多种处理方法

## 实践练习 - Practice Exercises

### 练习 1: 对比度增强
1. 找一张对比度很低的图像
2. 分别尝试直方图均衡化、对数变换、幂次变换
3. 对比哪种方法效果最好

### 练习 2: 噪声去除对比
1. 加载一张干净的图像
2. 添加椒盐噪声
3. 分别用均值滤波和中值滤波去噪
4. 对比哪种方法更好

### 练习 3: 边缘增强对比
1. 加载一张图像
2. 分别使用锐化滤波和高通滤波
3. 观察两者的区别

### 练习 4: 彩色图像处理
1. 加载彩色图像
2. 转换为灰度，观察效果
3. 重新加载，转换为HSV，观察各通道
4. 再次加载，使用彩色增强
